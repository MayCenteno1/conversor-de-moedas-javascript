## 📌 Sobre o projeto

Aplicação web simples e interativa para conversão de moedas, permitindo transformar valores em **Real (BRL)** para:

* Dólar (USD)
* Euro (EUR)
* Libra (GBP)

O projeto foi desenvolvido com foco em prática de fundamentos do **JavaScript**, manipulação do DOM e tratamento de eventos.
